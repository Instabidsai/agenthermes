# AgentHermes — Scope

## What AgentHermes IS
- Agent readiness scoring platform (9 dimensions, 6-step journey, 12 verticals)
- Business remediation (auto-generate agent cards, llms.txt, MCP tools, agent-hermes.json)
- Gateway/marketplace (hosted MCP per business, wallet billing, fulfillment routing)
- E-commerce adapters (Shopify, WooCommerce, Square auto-detection + MCP generation)
- Agent card registry (searchable directory of agent-ready businesses)
- Industry benchmark reports (State of Agent Readiness)
- The "Shopify of the Agent Economy" — make any business agent-ready

## What AgentHermes is NOT
- NOT a payment processor (Stripe/Visa/Mastercard handle that)
- NOT an agent framework (LangChain/CrewAI/AutoGen handle that)
- NOT building AI agents — we make businesses ready FOR agents
- NOT hosting customer websites or doing web development
- NOT an SEO consultancy (we score discoverability for agents, not search engines)
- NOT a CRM or business management tool
- NOT competing with MCP directories (Smithery, mcp.so) — we complement them

## Boundaries
- We score and remediate, we don't build customer products
- We host MCP endpoints, we don't host full applications
- We detect e-commerce platforms, we don't manage their stores
- We route fulfillment (email/webhook/lead), we don't process orders
- We propose standards (agent-hermes.json), we don't govern protocols
